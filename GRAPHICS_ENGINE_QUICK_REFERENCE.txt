================================================================================
WHAT WAS FIXED
================================================================================

OLD (BROKEN):
  [0:v]drawtext=...:[tmp0];[tmp0]format=yuv420p[v_out]
  ❌ Complex pad labels
  ❌ Semicolon separators
  ❌ Escaping issues

NEW (FIXED):
  drawtext=...,format=yuv420p
  ✅ Simple comma-separated
  ✅ Standard FFmpeg
  ✅ Reliable

================================================================================
FILTER SYNTAX EXAMPLES
================================================================================

SINGLE OVERLAY (Hook, 0-3 seconds):
  drawtext=text='Hook':fontsize=54:fontcolor=white:x=w/2:y=h/4:enable='between(t,0,3)',format=yuv420p

MULTIPLE OVERLAYS (Hook + Benefit + CTA):
  drawtext=text='Hook':fontsize=54:fontcolor=white:x=w/2:y=h/4:enable='between(t,0,3)',\
  drawtext=text='Benefit':fontsize=44:fontcolor=0x00d4aaff:x=w/2:y=h/2:enable='between(t,8,14)',\
  drawtext=text='CTA':fontsize=40:fontcolor=black:x=w/2:y=h*0.85:enable='between(t,41,45)',\
  format=yuv420p

================================================================================
FFMPEG COMMAND
================================================================================

Before (COMPLEX):
  ffmpeg -i input.mp4 \
    -filter_complex "[0:v]drawtext=...[tmp0];[tmp0]format=yuv420p[v_out]" \
    -map "[v_out]" -map 0:a \
    -c:v libx264 -preset medium -crf 23 -c:a copy -y output.mp4

After (SIMPLE):
  ffmpeg -i input.mp4 \
    -vf "drawtext=...,format=yuv420p" \
    -c:v libx264 -preset medium -crf 23 -c:a copy -y output.mp4

================================================================================
COMMON PARAMETERS
================================================================================

TEXT:
  text='Your text here'                    Text to display
  text='Don'\''t worry'                   Escaped apostrophe

POSITIONING:
  x=w/2                                   Center X
  y=h/4                                   1/4 down
  y=h*0.85                                85% down

TIMING:
  enable='between(t,0,3)'                 Show 0-3 seconds
  enable='between(t,2,5)'                 Show 2-5 seconds
  enable='between(t,41,45)'               Last 4 seconds of 45-sec video

COLORS:
  fontcolor=white                         Named color
  fontcolor=0x00d4aaff                    Hex RGB (RGBA)
  boxcolor=0x00000080                     Semi-transparent background

SIZE:
  fontsize=54                             Font size in pixels
  line_spacing=10                         Auto-calculated from font size

BOX BACKGROUND:
  box=1                                   Enable box
  boxcolor=0x00000080                     Box color
  boxborderw=2                            Border width

================================================================================
POSITIONING PATTERNS
================================================================================

CENTERED:
  x=w/2, y=h/2

TOP (HOOK):
  x=w/2, y=h/4

BOTTOM (CTA):
  x=w/2, y=h*0.85

LEFT:
  x=w*0.1, y=h/2

RIGHT:
  x=w*0.9, y=h/2

================================================================================
COLOR EXAMPLES
================================================================================

NAMED COLORS:
  white, black, red, green, blue, yellow, cyan, magenta

HEX COLORS (RGBA):
  0xFF0000FF - Red (fully opaque)
  0xFF000080 - Red (50% transparent)
  0x00d4aaff - Teal (money_blueprints brand)
  0x00000080 - Black (50% transparent)

BOX BACKGROUNDS:
  0x00000080 - Black, 50% transparent
  0x000000cc - Black, 80% transparent

================================================================================
TIMING PATTERNS
================================================================================

0-3 SECONDS (Hook):
  enable='between(t,0,3)'

2-5 SECONDS:
  enable='between(t,2,5)'

8-14 SECONDS (Benefit):
  enable='between(t,8,14)'

LAST 4 SECONDS (45-second video):
  enable='between(t,41,45)'

RELATIVE TIMING (Config format):
  timing_start_s: -4    # 4 seconds before end
  timing_end_s: 0       # Until end
