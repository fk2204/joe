TECHNICAL SPECIFICATIONS
================================================================================

Module Architecture:
  - graphics_engine.py: FFmpeg integration with overlay logic
  - overlay_templates.py: Python-based configs (fallback)
  - Config files: YAML with channel-specific styling
  - Integration points: Pipeline (Step 4b) and Uploader

Execution Flow:
  1. Load channels config from JSON
  2. Load/parse YAML overlay config
  3. Resolve template variables in overlay text
  4. Resolve timing (handle negative values)
  5. Build FFmpeg drawtext filters
  6. Chain filters into filter_complex
  7. Call FFmpeg with single-pass encoding
  8. Return output path or input path (non-fatal)

Error Recovery:
  - Missing YAML: use Python templates
  - Missing overlays: skip filtering
  - FFmpeg error: return input unchanged
  - All failures are logged for debugging

Performance Optimization:
  - Single-pass encoding (no two-pass waste)
  - Audio passthrough (no re-encode)
  - Minimal filter overhead
  - Lazy loading (parse YAML only when needed)
